# Gestione Libreria Videogiochi (Java)
Applicazione Java che permette di aggiungere, visualizzare e modificare videogiochi tramite un menu testuale. Il progetto utilizza classi, oggetti, incapsulamento e ArrayList.

## Funzionalità
- Aggiunta di un nuovo videogioco alla libreria
- Visualizzazione di tutti i giochi salvati
- Modifica dello stato (es. In corso, Completato, Da iniziare)
- Aggiornamento delle ore giocate
- Ricerca dei giochi tramite titolo
- Menu testuale semplice e intuitivo

## Struttura del progetto
- **Gioco** → rappresenta un singolo videogioco (titolo, genere, piattaforma, ore, stato)
- **LibreriaGiochi** → gestisce la lista dei giochi tramite `ArrayList`
- **GameManager** → gestisce il menu e l'interazione con l'utente
- **Main** → avvia il programma

## Come eseguirlo
Compilare ed eseguire la classe `Main`.  
Il programma si avvierà mostrando il menu principale.
